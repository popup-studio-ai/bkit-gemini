## 2. v1.5.6 New Features

### [UX-04] Level Policy Verification
**Prompt**:
```
Check if there are any policy TOML files in .gemini/policies/ directory.
```
**Expected**: 
- `bkit-starter-policy.toml` (or Dynamic/Enterprise) exists.
- Contains rules with `decision = "deny"` or `"ask_user"`.

### [UX-05] Tool Annotations Query
**Prompt**:
```
What are the annotations for the 'read_file' tool in tool-registry.js?
```
**Expected**: 
- `readOnlyHint: true`
- `destructiveHint: false`
- `idempotentHint: true`

### [UX-06] RuntimeHook SDK Status
**Prompt**:

**Expected**: 
- `mode = 'command'`
- `sdkAvailable = true` (if v0.31.0)
