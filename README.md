# bkit-gemini

> bkit Vibecoding Kit for Gemini CLI - PDCA + AI-native development

## Overview

bkit-gemini is a Gemini CLI extension that implements the PDCA (Plan-Do-Check-Act) methodology for AI-native development. This project is a port of [bkit-claude-code](https://github.com/anthropics/bkit-claude-code) for Gemini CLI.
