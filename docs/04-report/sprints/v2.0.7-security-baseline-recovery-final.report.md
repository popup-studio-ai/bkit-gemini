## 6. Lessons Learned

1. **AGENTS source-of-truth principle** — 21-agent 확장과 같은 큰 변경 시 derivative 데이터(policy groups, test expectations)가 자동 동기화되지 않으면 정적 grep 테스트가 silent하게 outdated 됨. 동적 회귀로 양방향 정합성을 강제하면 미래 한쪽만 수정해도 즉시 fail.
2. **Helper refactoring 후 static test cascade** — `_textResponse` helper 도입은 좋은 리팩토링이지만, 직접 `JSON.stringify` 패턴을 기대한 테스트가 모두 cascade fail. 정적 grep은 helper-aware로 작성하거나 동적 실행 검증으로 보완 필요.
3. **Sloppy mode false-negative** — Node.js의 frozen object 변경은 strict mode에서만 throw. tc80에 `'use strict'` 명시 후 frozen 검사가 진짜로 작동.
4. **모순처럼 보이는 두 테스트 (TC80-29 / TC91-25)는 보완적 의도** — "FULL agent 이름은 noted but no rules"는 toml footer comment로 만족 가능.
