## 8. Lessons Learned

1. **사람 / LLM 컨텍스트 분리** — GEMINI.md(LLM)는 슬림 유지, README(사람)도 인덱스만. 깊이 있는 reference는 별도 디렉터리 (`docs/reference/`)에서 on-demand 로드.
2. **Re-bloat 방지를 동적 회귀로 강제** — 각 reference 파일 ≤ 25KB 상한 (OB-05). 미래 누가 README 재비대화 시도하면 즉시 fail.
3. **Legacy anchors hint** — 외부 블로그 link 깨짐 우려에 대해 README 하단에 "Legacy anchors moved to docs/reference/" 명시 + 각 reference 파일 첫 줄에 출처 명시. 검색 엔진 link도 자연스럽게 정착.
4. **Section mapping 표는 reference 파일의 첫 줄에 보존** — git mv 없이 cut/paste로 분할했지만, 각 reference 파일 헤더에 "Moved from README.md during v2.0.7-S4" 명시로 history 추적 가능.
