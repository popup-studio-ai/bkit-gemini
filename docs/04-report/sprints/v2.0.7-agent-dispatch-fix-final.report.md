## 7. Lessons Learned

1. **Sync hooks + MCP**: Gemini CLI's synchronous BeforeModel hook cannot directly invoke MCP tools. The directive-injection pattern (model receives explicit JSON args + boundary instruction) is portable across all CLI versions ≥ v0.36.0 and degrades gracefully when async hooks land.
2. **Filesystem registry > hardcoded list**: `fs.readdirSync(AGENTS_DIR)` gives one source of truth — adding a new agent to `agents/*.md` automatically grows the whitelist with no code change.
3. **DUAL-test surface**: Same matcher exercised at three layers (unit, before-model integration, full 168-matrix) catches drift at every boundary. AD-01~22 catches regex bugs; BM-01~12 catches integration regressions; FM catches lang/agent cartesian regressions.
4. **Cross-OS pure JS**: No native bindings, no shell out, no bash parser dependency — runs identically on macOS / Windows / Linux / WSL / Git Bash. Reaffirms D16' decision from S7 RCA.
5. **Drift detection via baseline measurement**: Pre/Post AC-T6 with stash gave exact regression count (0). TC110-01 8→11 truth update was identified as incidental, justifying inclusion in S3 scope.
