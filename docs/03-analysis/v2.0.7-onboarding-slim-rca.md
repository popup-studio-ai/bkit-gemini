### 3.2 정보 무손실 보증

- README 모든 섹션의 내용은 **새 reference 파일들에 100% 보존** (단순 cut/paste + 인덱스 trail)
- 외부 링크 anchor가 README의 특정 섹션을 가리킬 가능성 → 압축된 README 하단에 "Legacy anchors moved to docs/reference/" + 주요 anchor stub 유지
- GEMINI.md에 docs/reference/ 포인터 추가 (LLM이 deep reference 접근 시)

### 3.3 Out-of-scope (정당화)

- **신규 사용자 실측 onboarding 시간 (30분→5분 검증)**: 사용자 환경 / 첫 사용자 모집 필요. framework PoC + 명확한 5분 가이드 작성으로 입증.
- **외부 블로그/문서 링크 자동 redirect**: 308 redirect는 호스팅 환경 (GitHub은 미지원). README hash anchor stub으로 mitigation.
